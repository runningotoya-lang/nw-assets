Privacy Policy

Last updated: July 3, 2026

NW-post-helper ("the Tool") is a private, internal-use application. This policy explains how the Tool handles data.

1. Data Collected

The Tool stores TikTok API access tokens for the operator's own TikTok accounts, and the photo files and captions the operator prepares for posting. No data is collected from any third party or from any other TikTok user.

2. Use of Data

Stored tokens are used only to send photo drafts to the operator's own TikTok accounts via TikTok's official Content Posting API. Content files are hosted temporarily for transfer to TikTok and removed after publication.

3. Data Sharing

No data is sold, shared, or disclosed to any third party, except transmission to TikTok's official API as required for the Tool's function.

4. Data Retention and Deletion

Tokens and content are stored on the operator's own local machine and can be deleted by the operator at any time.

5. Contact

For privacy inquiries, contact the operator via the email address registered with the associated TikTok developer account.
